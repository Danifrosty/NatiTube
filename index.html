Overview

NatiTube is a code-injection project designed to bypass the limitations of the YouTube mobile web player. It isolates the HTML5 video stream and delivers it directly to the operating system's native media controller (AVPlayer for iOS and MediaPlayer for Android).

True Multitasking

By using the native system player, Picture-in-Picture mode integrates perfectly with your mobile OS window manager, allowing you to use other apps without playback interruption.

System Integration

Native controls provide access to system-level gestures for volume, brightness, and AirPlay/Cast features that are often restricted in web browsers.

How It Works

The core logic utilizes a DOM observation loop that monitors m.youtube.com for the <video> element. Once active, the script performs three critical operations:

  • DOM Re-parenting: Moves the video element to the top of the document hierarchy to prevent YouTube's scripts from overriding its state.
  • Control Enforcement: Triggers the controls = true attribute, forcing the browser to render the native OS UI instead of the custom YouTube overlay.
  • CSS Dominance: Applies high-priority styling to ensure the player maintains the correct aspect ratio and visibility above all other web elements.

Installation Guide

Android Setup

1. Install a UserScript manager (Kiwi Browser, Orion, or InjecThor).

2. Load the android-version.js script from the repository.

3. Ensure the target URL is set to *://m.youtube.com/*.

iOS Setup

1. Access the official Shortcut link via the iOS-url file.

2. Enable "Allow Untrusted Shortcuts" in your device settings if required.

3. Execute the shortcut from the Safari Share Sheet while on a YouTube page.

Frequently Asked Questions

Why is the video quality low?

YouTube uses adaptive bitrate streaming (DASH). When switching to the native player, the browser may default to a lower resolution stream. It is recommended to manually select your desired quality in the YouTube settings before activating the script.

Does it work with the YouTube App?

No. NatiTube is a web-based injection tool designed specifically for mobile browsers. It relies on the browser's ability to render native HTML5 video controls.

Privacy Note: NatiTube does not collect data, connect to external servers, or track your history. The code is 100% open-source and executes locally on your hardware.
