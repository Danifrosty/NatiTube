NatiTube

Hybrid Space Layer engine for high-performance native video playback.

Overview

NatiTube is an advanced code-injection project designed to replace the standard YouTube mobile web UI with a high-priority hardware-accelerated native layer. It maintains the platform's core social features while optimizing visual delivery.

Hybrid Stream Bridge

NatiTube captures raw video frames via the captureStream() API and pipes them into a master native overlay for zero-latency playback.

Space Layout Mode

The interface uses absolute positioning to push comments and descriptions below the player, creating a non-intrusive viewing environment.

Installation Guide

Standard Deployment

1. Install a UserScript manager such as Tampermonkey, Violentmonkey, or Kiwi Browser.

2. Create a new script and paste the natitube.js code from the repository.

3. Navigate to YouTube and the Space Layer will activate automatically on video pages.

Security & Privacy

Architecture Note: NatiTube operates entirely on the client side. No data is transmitted to external servers. The bridging process is isolated to the local browser instance.
